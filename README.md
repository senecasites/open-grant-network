# EcoServants® Open Grant Network

**Purpose:**  
This repository contains all documentation, ethical guidelines, and technical resources for the **EcoServants® Open Grant Network (OGN)** — a nonprofit initiative to make grant data transparent, lawful, and accessible.

---

---

## Collaboration Guide

1. **Create a new branch** using your name and focus (example: `latha-phase0-docs`).  
2. **Add or edit files** inside your assigned folder (`/docs` or `/dev`).  
3. **Commit changes** with clear messages (example: `Add draft Ethical Crawling Guidelines v1`).  
4. **Submit a Pull Request (PR)** to the Data Team Lead before merging to `main`.  

---

## File Naming Convention

Use this pattern for consistency:

YYYY-MM-DD_filename.ext

Example: `2025-10-19_Ethical_Guidelines.md`
